# Bioclimática Paraguay 🌿

Herramienta web interactiva de **diseño bioclimático** para arquitectos, constructores y estudiantes en Paraguay.

Incluye guía por pasos, zonificación climática, estrategias pasivas, base de materiales y plantas nativas, y editor isométrico 3D.

## 🧭 Secciones

- **Guía** — Wizard paso a paso: seleccioná ubicación en el mapa, orientación del lote, tipo de edificio y recibí recomendaciones personalizadas con diagramas.
- **Zonas** — 4 zonas climáticas del Paraguay con estadísticas, rosa de vientos, trayectoria solar y precipitaciones.
- **Estrategias** — 19 estrategias bioclimáticas pasivas filtrables por categoría.
- **Materiales** — 20 materiales de construcción con propiedades térmicas (λ, masa, aislamiento).
- **Plantas** — 27 especies nativas del Paraguay con usos bioclimáticos, filtro por tipo y ubicación.
- **Herramientas** — Calculadora de resistencia térmica (R/U), calculadora de captación de agua de lluvia, y comparador de zonas climáticas.

## 🛠️ Herramientas incluidas

| Herramienta | Descripción |
|---|---|
| 🧱 Resistencia térmica | Calculá R y U de muros multicapa |
| 💧 Captación de lluvia | Estimá litros/año según zona y techo |
| 📊 Comparador de zonas | Compará 2 zonas side-by-side |

## 🎨 Temas visuales

- **Arquitectónico** — limpio, técnico
- **Blueprint** — plano clásico azul
- **Cyberpunk** — neon / oscuro
- **Paper** — boceto en papel

## 📐 Editor isométrico

Editor 3D en el navegador para diagramar lotes, volúmenes, árboles y sombras con 4 vistas de cámara.

## 🚀 Cómo usar

Abrí `index.html` en cualquier navegador o serví la carpeta:

```bash
npx serve .
```

## 🌳 Stack

Vanilla JS · Leaflet · Canvas API · CSS3 · Sin dependencias externas (solo Leaflet para mapas)
